Para fazer o projeto funcionar, criar um banco de dados chamado "calendar" no banco de dados e rodar a sql dentro do arquivo queries.sql.
As informações de conexão com o banco de dados estão no arquivo ConnDB.php.

Para rodar, basta jogar a past do projeto e acessar via localhost/<pasta raiz>.

1) Clique em um dia e dê uma descrição, e envie.
    - A página será atualizada.
2) Clique no dia referido para ver a descrição dele.

Só é possível ter 1 descrição por dia.
